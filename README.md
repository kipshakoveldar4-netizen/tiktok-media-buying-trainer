# TikTok Media Buying Trainer

MVP веб-приложения для подготовки к TikTok Media Buying Certification.

## Стек

- Next.js
- TypeScript
- Tailwind CSS
- App Router

## Возможности

- Главная страница с описанием приложения.
- Раздел обучения с конспектами по каждой теме.
- Локальный AI-чат по базе знаний без OpenAI API.
- Прогресс пользователя в localStorage: изученные темы, результаты, лучший экзамен и слабые темы.
- Страница тем с 11 разделами сертификации.
- Тематический тест после выбора темы.
- Экзаменационный режим на 20 случайных вопросов.
- Результат в процентах.
- Просмотр правильных и неправильных ответов.
- Отдельная страница разбора ошибок.
- Кнопка повторного прохождения.

## Запуск

```bash
npm install
npm run dev
```

После запуска откройте `http://localhost:3000`.
