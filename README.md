# TikTok Media Buying Trainer

MVP веб-приложения для подготовки к TikTok Media Buying Certification.

## Стек

- Next.js
- TypeScript
- Tailwind CSS
- App Router

## Возможности

- Главная страница с описанием приложения и блоком прогресса.
- Простая система доступа по коду без базы данных.
- Демо-режим по умолчанию: 20 вопросов, ограниченное обучение и ограниченный AI-чат.
- Полный доступ после ввода кода `TIKTOK2026`.
- Сохранение доступа и прогресса в localStorage.
- Раздел обучения с конспектами по темам сертификации.
- Локальный AI-чат по базе знаний без OpenAI API.
- Страница тем с 11 разделами сертификации.
- Тематические тесты после выбора темы.
- Экзаменационный режим на 20 случайных вопросов.
- Результат в процентах, правильные и неправильные ответы.
- Разбор ошибок и слабые темы.
- Кнопка повторного прохождения и сброс прогресса.

## Запуск

```bash
npm install
npm run dev
```

После запуска откройте `http://localhost:3000`.
